# Multimodal Emotion Recognition

## Project Overview
This project detects human emotions using:
- Speech
- Text
- Speech + Text Fusion

## Dataset
TESS (Toronto Emotional Speech Set)

## Technologies Used
- Python
- Librosa
- Scikit-learn
- TF-IDF
- Random Forest

## Models
1. Speech Emotion Recognition
2. Text Emotion Recognition
3. Multimodal Fusion Emotion Recognition

## Accuracy
- Speech Model: 100%
- Text Model: 100%
- Fusion Model: 100%

## Installation

Clone repository:

```bash
git clone https://github.com/shravanikammari01/Multimodal-Emotion-Recognition
```

Move into project folder:

```bash
cd Multimodal-Emotion-Recognition
```

Install dependencies:

```bash
pip install -r requirements.txt
```
